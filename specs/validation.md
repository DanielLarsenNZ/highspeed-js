# Validation

## Players
> A game has one to four players. Player numbers are zero-based. Constants define domain specific language for Player numbers.

These are valid numbers for specifying a player:

* PLAYER_1
* PLAYER_2
* PLAYER_3
* PLAYER_4
* 0
* 1
* 2
* 3

These are not valid numbers:

* 4
* -1
* null
* "1"
* undefined
